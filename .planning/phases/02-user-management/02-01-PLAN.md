## Research-Derived Patterns

From 02-RESEARCH.md:
- Use `adduser` (not `useradd`) on Ubuntu
- Use `openssl rand -base64 32` for passwords
- Use `install -d -m 700` for .ssh directory
- Use `chpasswd` for non-interactive password setting
- Check-before-create idempotency: `id "$user" &>/dev/null`
- Check-before-append for SSH keys: `grep -qF "$key" "$file"`
