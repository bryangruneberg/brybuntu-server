```bash
check_root
log_info "Configuring sudo access for amazeeio..."
sudoers_create_nopasswd "amazeeio"

```

### Library Functions Available

**lib/user.sh - user_create_with_ssh(username, ssh_key)**
- Creates user with adduser if not exists
- Generates random password with openssl rand -base64 32
- Creates ~/.ssh directory with 700 permissions
- Adds SSH key to authorized_keys with 600 permissions
- Idempotent: checks before creating, skips if already exists

**lib/sudo.sh - sudoers_create_nopasswd(username)**
- Creates /etc/sudoers.d/{username} with "{username} ALL=(ALL) NOPASSWD: ALL"
- Validates syntax with visudo -c BEFORE installing (prevents root lockout)
- Sets permissions to 440
- Idempotent: checks if file exists with correct content

**lib/dev.sh - install_lazyvim_for_user(username)**
- Clones LazyVim starter to ~/.config/nvim/
- Removes .git folder to make it user's own config
- Sets proper ownership
- Checks if Neovim is installed first
- Idempotent: skips if config directory already exists
