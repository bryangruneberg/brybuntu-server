## BuildKit Decision

Per Claude's discretion (08-CONTEXT.md):
- Configuration approach: Use /etc/profile.d/docker-buildkit.sh for DOCKER_BUILDKIT=1
- buildx verification: Modern Docker includes buildx — verify it's available
- Alternative: daemon.json "features": {"buildkit": true} — but env var is simpler and more explicit

## Implementation Notes

- buildx is included with Docker Engine 19.03+ (we're installing latest)
- DOCKER_BUILDKIT=1 enables BuildKit for docker build commands
- Environment variable approach is cleaner than daemon.json modification
- Profile.d script ensures all users get the setting
