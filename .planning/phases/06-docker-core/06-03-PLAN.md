### Technical Details
From 20-users/15-dgxc.sh:
- Use id command to check if user exists
- usermod -aG docker $user for group add
- Install or check with getent group docker

Systemd pattern:
- systemctl enable --now docker
- Enables auto-start and starts now
- Check: systemctl is-enabled docker

Idempotency:
- Check user in group: groups $user | grep -q docker
- Skip if already member
- For systemd: check before enable
