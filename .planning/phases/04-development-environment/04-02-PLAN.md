## Implementation Details

```bash
# Source guard

# Functions:
# - install_lazyvim_for_user(username)
```

**Key Features:**
- Source guard pattern (prevents double-sourcing)
- Depends on lib/common.sh for logging
- Reusable per-user installation function

### Module: `modules/40-dev/30-neovim.sh`

Installs Neovim via AppImage extraction:

1. Check if correct version already installed
2. Download Neovim AppImage (v0.11.6)
3. Extract AppImage to `/opt/nvim-${VERSION}/`
4. Create symlink at `/usr/local/bin/nvim`
5. Verify installation

**Key Points:**
- AppImage extraction avoids building from source
- System-wide installation in /opt/
- Version-pinned for reproducibility
- Idempotent: checks version before reinstalling

### Module: `modules/40-dev/40-lazyvim.sh`

Configures LazyVim for both users:

1. Verify git is installed
2. Verify Neovim is installed
3. Call `install_lazyvim_for_user "bryan"`
4. Call `install_lazyvim_for_user "amazeeio"`

**Key Points:**
- Uses lib/dev.sh for reusable function
- Per-user configuration in `~/.config/nvim/`
- Idempotent: skips if config already exists
- Preserves user customizations on re-run
