## Current State

## Goal

Install Node.js LTS and Opencode CLI globally so both users can immediately use these tools after SSH login.

## Implementation Details

### Module: `modules/40-dev/10-node.sh`

Installs Node.js LTS via NodeSource repository:

1. Check if Node.js already installed (idempotency)
2. Install dependencies (curl, ca-certificates, gnupg)
3. Download and execute NodeSource setup script
4. Install Node.js and npm
5. Verify installation

**Key Points:**
- Uses NodeSource LTS setup script
- Idempotent: checks for existing installation
- System-wide installation accessible to all users

### Module: `modules/40-dev/20-opencode.sh`

Installs Opencode CLI via direct binary download from GitHub releases:

1. Check if opencode already installed
2. Install dependencies (curl, tar)
3. Detect target platform (architecture, AVX2 support)
4. Get latest version from GitHub API
5. Download and extract binary from GitHub releases
6. Move binary to /usr/local/bin/
7. Verify installation

**Key Points:**
- Direct binary download from GitHub releases
- No dependency on npm/Node.js (independent installation)
- Supports x64 and arm64 architectures
- Handles baseline builds for CPUs without AVX2
- Idempotent: checks for existing binary
